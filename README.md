# Personal Portfolio Website

Built during internship at Internee.pk

## Features
- WordPress + Elementor
- Dynamic blog with categories
- Contact form + Google reCAPTCHA
- Dark/Light mode toggle
- Custom JavaScript animations
- AI Chatbot (Gemini API)

## Technologies
- WordPress 6.9.4
- Elementor (Free)
- PHP, JavaScript, CSS

## Local Setup
1. Install LocalWP
2. Clone this repo into `/wp-content/themes/`
3. Activate theme in WordPress
